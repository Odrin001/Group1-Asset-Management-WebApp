# SDCA Asset Management System

A simple web app for managing school assets.

---

# Features

* Login, Register, and Forgot Password pages
* Clean UI with Tailwind CSS
* Green theme design
* Built using Next.js (App Router)
* Uses TypeScript

---

# Pages

* **Login** (`/login`) – sign in using school email
* **Register** (`/register`) – create an account
* **Forgot Password** (`/forgot-password`) – reset password

---

# Project Structure

```
app/
 ├── (auth)/
 │   ├── login/
 │   ├── register/
 │   └── forgot-password/
 ├── globals.css
 └── layout.tsx
```

---

# How to Run

1. Install dependencies:

```
npm install
```

2. Run the project:

```
npm run dev
```

3. Open in browser:

```
http://localhost:3000/login
```
